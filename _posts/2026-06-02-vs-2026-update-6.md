Visual Studio 2026 Update 6 is now available. This update includes several improvements and bug fixes for the Visual C++ compiler and tools known as the [MSVC Build Tools 14.51](https://devblogs.microsoft.com/cppblog/msvc-version-1451-available/) release including a matching [STL](https://github.com/microsoft/STL/wiki/Changelog#msvc-build-tools-1451) release. For a summary of changes for 18.1 to 18.6, see the [Visual C++ Team Blog](https://devblogs.microsoft.com/cppblog/whats-new-for-c-developers-in-visual-studio-2026-18-1-18-6/).

<!--more-->

VS 2026 18.6 includes a new version of the C/C++ Compiler (19.51.36243). The ``v145`` Platform Toolset includes support for C++14, C++17, C++20, and C11/C17. A preview of C++23 support is available via `/std:c++23preview`. C++26 support for the Standard Library is also in progress in this release using `/std::c++latest`. See [Microsoft Learn](https://learn.microsoft.com/cpp/overview/visual-cpp-language-conformance) for more details on the C++ language and library conformance status. Note that C11/C17 support requires Windows SDK (20348) or later. For a step-by-step guide on upgrading projects, see [this blog post](https://devblogs.microsoft.com/cppblog/upgrading-c-projects-to-visual-studio-2026/).

18.x update | C++ compiler
--|--
18.6.0|19.51.36243.0

<strong>C++23:</strong> For details on improved C++23 support, see [this blog post](https://devblogs.microsoft.com/cppblog/c23-support-in-msvc-build-tools-14-51/). Some C++23 features are still to be implemented, and are planned for a future update.

<strong>CMake:</strong> VS 2026 Update 6 ships with [CMake 4.2.3](https://cmake.org/cmake/help/v4.2/release/4.2.html) which was added in Update 4. VS 2026 Update 7 ships with [CMake 4.3.1](https://cmake.org/cmake/help/v4.3/release/4.3.html).

<strong>Clang:</strong> VS 2026 RTM - Update 6 ships with clang v20.1.8. VS 2026 Update 7 ships with clang v22.1.3.
